### 🚀 Quick Theme Install

```bash
# Install all 33 themes
./scripts/install-all-themes.sh

# Launch interactive theme selector
./scripts/theme-selector.sh

# Or apply a theme directly
source ~/.config/terminal-themes/vampires-letter.sh
```
